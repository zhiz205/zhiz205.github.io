咖啡的由來

我們喝的咖啡是怎麼來的呢？

首先從咖啡豆開始介紹，可以參考下面的相關資訊，另外還有提供咖啡喜好的測驗和菜單的介紹，也許下次在咖啡廳就不會無從選擇了！

~ ~ Introduce ~ ~

品種

種類有阿拉比卡、羅布斯塔

產地

大致分布於非洲、南美洲等國家

處理法

利用日曬或水洗等處理方法

烘焙程度

大致分為淺焙、中焙、深焙

常見品種

  • 阿拉比卡
    • 生長在西半球的高原地區（蟲害少 收成多）
    • 較羅布斯塔豆味道甜、柔和、酸、咖啡因低
  • 羅布斯塔
    • 綠原酸含量高，不易受蟲害氣候影響
    • 生長在地勢低較炎熱地區
    • 風味較苦澀強烈
    • 咖啡因高

常見產地

  • 衣索比亞
    • 阿拉比卡豆原產地
    • 有糖漿味、明顯的酸性與果香氣味
    • 以濃郁、樸實的咖啡聞名
  • 耶加雪菲
    • 原為西達摩產區，但因太具辨識性而獨立出來
    • 風味有柑橘、莓果、花香、茶感
  • 肯亞
    • 有糖漿、黑醋栗的芬芳
    • 酸度明顯
  • 哥斯大黎加
    • 完美結合乾淨的味道、香氣、適中的酸度，因此成為世界上最好的咖啡之一
    • 標有SHB高海拔嚴格硬度
    • 只允許種阿拉比卡咖啡豆，種植羅布斯塔豆是非法的
  • 瓜地馬拉
    • 風味：活潑、辛辣、複雜
    • 以顆粒特大的「大象」咖啡豆聞名
  • 牙買加
    • 以顆粒大 品質佳 味道調和 適當的酸苦香醇甜的「藍山咖啡」聞名
    • 是全世界公認的極品咖啡
  • 哥倫比亞
    • 僅次巴西待遇世界第二大咖啡生產地
    • 只種阿拉比卡咖啡豆
  • 印尼
    • 大膽的風味：泥土味、苦味、香料味、木材味、菸草味、皮革味等（類似不加糖的黑可可）
    • 常用來做混合豆使用
    • 以阿拉比卡原生種的「曼特寧」、安拉卡聞名
  • 越南
    • 為羅布斯塔豆為主，以阿拉比卡豆為輔來增加利潤
    • 味道濃烈、有苦味、酸度低、咖啡因高
    • 常用來做混合豆與即溶咖啡

處理法

  • 日曬
    • 步驟：篩選浮果 -> 曝曬 -> 脫殼去除果皮果肉
    • 風味：酸度 苦度較柔和、甜度果香酒香明顯
  • 水洗
    • 步驟：篩選浮果 -> 去除果皮果肉 -> 泡水發酵去除果膠 -> 清洗 -> 乾燥 -> 脫殼
    • 風味：酸度強烈
    • 口感：清爽乾淨
  • 半水洗
    • 步驟：篩選浮果 -> 去除雜質果皮果肉 -> 去除果膠 -> 乾燥脫殼 -> 二次乾燥
    • 與水洗相比甜味較高 酸味低 口感沈穩
  • 蜜處理
    • 步驟：篩選浮果 -> 去除果皮果肉 -> 曝曬 -> 脫殼
    • 甜味酸度低 帶有果香
  • 濕刨法
    • 步驟去除果皮 -> 短時間曝曬乾燥到含水量約35% -> 去除外殼 -> 曬乾到不易腐壞的程度
    • 特殊的青草藥 香草風味
    • 外表顏色為綠色

烘焙程度

  • 淺烘焙(Light Roast)
    • 經過短時間的烘焙處理，咖啡生豆中水分還沒完全蒸發
    • 豆子會比較重也比較硬
    • 咖啡因相對較高
    • 保留咖啡豆的原始的風味
    • 通常帶有花香或果香，酸度明顯而苦味較低，口感清新
    • 很適合以手沖方式來沖煮
    • 適合喜歡爽口、沒有厚重苦味咖啡的人群
  • 中烘焙(Medium Roast)
    • 經過適中的烘焙時間
    • 豆子呈現出中棕色，表面略帶油光
    • 保留一部分原始的咖啡果實風味
    • 除了會帶有焦糖、巧克力的醇厚與微苦外，也能感受到淡雅的果香氣息
    • 咖啡酸苦適中，能同時感受到咖啡的香、酸、醇、苦
    • 大眾接受度最高的咖啡種類
  • 深烘焙(Dark Roast)
    • 經過長時間的高溫烘焙
    • 豆子呈現出深棕色甚至接近黑色，表面有明顯的油光
    • 咖啡因含量相對較低
    • 酸度幾乎完全消失，取而代之的是濃厚的苦味，擁有焦苦風味與濃郁厚實的口感
    • 通常用於Espresso咖啡、拿鐵、美式
    • 適合喜歡重口味咖啡的人群
